GET /api/youtube-channel-counter/user/:id

Returns detailed statistics for a YouTube channel using its Channel ID (e.g., UCX6OQ3DkcsbYNE6H8uQQuVA).

The response includes ytstudio (estimated real-time subscriber count) and statistics (detailed breakdown including views, videos, and API values).

GET /api/requests

Returns the total number of requests made to the API (Simple format).


{
  "totalapirequests": 20224594
}
